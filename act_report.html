twitter_archive_master = pd.read_csv("twitter_archive_master.csv",encoding="utf-8", dtype=dtypes,parse_dates=['timestamp'])

I observer athat above dataset could be considered as following parts:-

unique features:

  • tweet_id
  • timestamp
  • text
  • expanded_urls
  • jpg_url

features those could be used as a category:

  • source
  • type
  • stage

features with numeric values:

  • rating_numerator
  • rating_denominator
  • retweet_count
  • favorite_count

others remaining features:

  • name
  • in_reply_to_status_id
  • in_reply_to_user_id

And this part would help us to apply proper analizing tool on the given dataset.
